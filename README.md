# 🏋️ Workout Planner - 膝盖调整版训练计划

一个带计时功能的训练计划网页应用，专为膝盖（半月板）损伤设计。

## 功能

- ⏱️ 自动计时 + 组间休息
- 🔊 倒计时/开始/暂停蜂鸣提示
- 📊 进度条 + 组数/轮次显示
- 📝 训练日志记录
- ✏️ 在线编辑训练计划
- 📥📤 JSON 导入/导出
- 💾 localStorage 自动保存进度
- 🔄 循环训练支持（多轮）
- 📱 手机友好

## 训练安排

| 日 | 内容 |
|----|------|
| 周一 | 上肢推力日 |
| 周三 | 下肢日 |
| 周四 | 背+核心日 |
| 周六 | 全身循环（3轮） |

## 使用

直接打开 `index.html` 或访问 GitHub Pages。

### 自定义计划

1. 点击「✏️ 编辑」按钮修改计划
2. 点击「📤 导出」下载 JSON 文件
3. 点击「📥 导入」加载 JSON 文件
